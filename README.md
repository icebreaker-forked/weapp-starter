# Weapp Starter

![logo](./public/logo.png)

A minimal WeChat Mini Program starter template with Vite and Vue 3 SFC.

## Quick Start

### Development
```bash
pnpm dev              # Start dev server
pnpm dev --open       # Start and open WeChat DevTools
```

### Build
```bash
pnpm build
pnpm open             # Open WeChat DevTools
```

## Features

- **Vue 3 SFC**: Write components with `<script setup>` syntax
- **Vite**: Fast build tool and dev server
- **Wevu**: Vue components for mini programs
- **JSON Configuration**: Use `<json>` blocks for page config
